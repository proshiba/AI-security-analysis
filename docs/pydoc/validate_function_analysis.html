case集合の代表関数静的解析と全体ロジックが完了条件を満たすか検証する。
 
公開成果物だけを読み、検体、Ghidra project、生の逆コンパイル本文には触れない。
発見関数のinventoryと代表関数の選定理由が記録され、代表関数の逆コンパイルまたはCIL解析が試行され、
制約付きの結果には理由と次の解析方針が残っていることを確認する。
